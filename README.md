# CRM Masterpiece (FUTURE_FS_2)
A professional, secure, and responsive Lead Management Dashboard built to track business contacts and sales status.[ i have created a login page if you want to login {use mailid: crm123@gmail.com} {password:future}
## Features
* **Secure Authentication:** User login protected by Supabase Auth.
* **Real-time Database:** Add, view, and delete leads stored in a secure Supabase database.
* **Row-Level Security (RLS):** Database access is restricted to authenticated users only.
* **Responsive UI:** Modern design using Inter font and custom CSS for a clean look.

## Technologies Used
* **Frontend:** HTML5, CSS3, JavaScript.
* **Backend:** [Supabase](https://supabase.com/) (Database & Authentication).
* **Hosting:** GitHub Pages.

## Getting Started
1. Clone the repository: `git clone https://github.com/yourusername/FUTURE_FS_2.git`
2. Open `index.html` in your browser.
3. Use your configured Supabase credentials to access the dashboard.

## Security
This project uses Supabase Row Level Security (RLS) to ensure that only authorized users can access the leads table.
